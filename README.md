# Support Ticket Classification System
## Project Description
This project is a Machine Learning based Support Ticket Classification System developed using Python and Scikit-learn. The system automatically classifies customer support tickets into different categories and assigns priority levels based on ticket content.
## Features
- Support Ticket Classification
- Priority Level Detection
- NLP Text Processing
- Machine Learning Prediction
- Model Accuracy Evaluation
## Technologies Used
- Python
- Scikit-learn
- NLP
- VS Code
## Dataset
The project uses sample customer support tickets for training and testing the machine learning model.
## Working Process
1. Collect support ticket data
2. Preprocess text data
3. Train machine learning model
4. Predict ticket category
5. Assign priority level
6. Display output results
## Example Input
Payment failed during transaction
## Example Output
Ticket Category : Payment  
Priority Level : High
## Model Accuracy
Accuracy : 33.33%
## How to Run
1. Install Python
2. Install required library:
pip install scikit-learn
3. Run the program:
python support_ticket.py
